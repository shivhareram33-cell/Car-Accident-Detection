## Real time car accident detection model

A Deep learning model that is able to find car accident in real time using a custom-trained model. in the uploaded images and videos model finds accident scenes and highlights detected objects with bounding boxes to support intelligent traffic monitoring.

## Project Overview

Road accident are a major cause of injuries and fatalities worldwide.
this project using Deep learning, enabling faster identification of accident.

this project is a prototype of a advance project that can be used in aasisting emergency response and send emergency alert to near police and ambulance.

The application is built using Python, Flask , OpenCV and Computor Vision and provides an easy-to-use web interface for accidet detection.


## Features-

* Real time Car Accident Detection
* Real time vehicle tracking
* Image/Vedio based accident detection
* Easy interface
* Bounding Box Visualization
